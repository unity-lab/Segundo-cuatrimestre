1. crear entorno virtual dentro del directorio backend
   python -m venv .venv

2. ejecutar entorno virtual (activar)
    .venv\Scripts\activate

3. instalar dependencias o librerias en python para ese entorno virtual, se ejecuta en el nivel 
donde esta el .txt
    pip install -r .\requirements.txt

4. ejecutar el comando para correr el server de fastapi (libreria api de python simple), este comando se ejecuta
en el nivel donde se encuentre el archivo main.py  (en este caso suelto en dir backend)
    uvicorn maim:app  

5. con el comando ctrl + c en la parte de la terminal se detiene la ejecucion del servidor  

6. Renombrar .env.dev a .env y luego colocar las credenciales que deseen
    docker-compose y config_db.py van a tomar estas variables de entorno para el funcionamiento
    del sistema.
